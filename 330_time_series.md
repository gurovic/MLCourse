### **2.1 –ê–≤—Ç–æ—Ä–µ–≥—Ä–µ—Å—Å–∏—è (AR) –∏ –º–æ–¥–µ–ª—å ARIMA**  
**ARIMA (p, d, q):**  
- **p** ‚Äî –ø–æ—Ä—è–¥–æ–∫ –∞–≤—Ç–æ—Ä–µ–≥—Ä–µ—Å—Å–∏–∏ (–∑–∞–≤–∏—Å–∏–º–æ—Å—Ç—å –æ—Ç –ø—Ä–æ—à–ª—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π)  
- **d** ‚Äî –ø–æ—Ä—è–¥–æ–∫ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏—è (—É—Å—Ç—Ä–∞–Ω–µ–Ω–∏–µ —Ç—Ä–µ–Ω–¥–∞)  
- **q** ‚Äî –ø–æ—Ä—è–¥–æ–∫ —Å–∫–æ–ª—å–∑—è—â–µ–≥–æ —Å—Ä–µ–¥–Ω–µ–≥–æ  

```python
from statsmodels.tsa.arima.model import ARIMA  

model = ARIMA(data['temperature'], order=(1, 1, 1))  
model_fit = model.fit()  
forecast = model_fit.forecast(steps=5)  # –ø—Ä–æ–≥–Ω–æ–∑ –Ω–∞ 5 –¥–Ω–µ–π  
print(forecast)  
```

### **2.3 LSTM –¥–ª—è –ø—Ä–æ–≥–Ω–æ–∑–∏—Ä–æ–≤–∞–Ω–∏—è –≤—Ä–µ–º–µ–Ω–Ω—ã—Ö —Ä—è–¥–æ–≤**
